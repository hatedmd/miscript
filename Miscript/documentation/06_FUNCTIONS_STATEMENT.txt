FILE: functions/statement_parser.py
PURPOSE: Parse all statement types from .mis files with nested parentheses support

================================================================================
                         STATEMENT TYPES
================================================================================

Type: 'print'
-------------
Example: print("hello")
Structure:
    {
        'type': 'print',
        'argument': '"hello"',
        'line': 1
    }

Type: 'input'
-------------
Example: name = input("Enter name > ")
Structure:
    {
        'type': 'input',
        'var_name': 'name',
        'prompt': '"Enter name > "',
        'line': 1
    }

Type: 'window_init'
-------------------
Example: window = window.init(600, 600)
Structure:
    {
        'type': 'window_init',
        'var_name': 'window',
        'width': 600,
        'height': 600,
        'line': 1
    }

Type: 'window_method'
---------------------
Example: window.Button(onClick:function(), x:100, y:100)
Structure:
    {
        'type': 'window_method',
        'window_name': 'window',
        'method': 'Button',
        'params': {
            'onClick': {'value': 'function', 'type': 'function_call'},
            'x': {'value': 100, 'type': 'number'},
            'y': {'value': 100, 'type': 'number'}
        },
        'line': 1
    }

Type: 'function_start'
----------------------
Example: function(){
Structure:
    {
        'type': 'function_start',
        'body': []
    }

Type: 'function_end'
--------------------
Example: }
Structure:
    {
        'type': 'function_end'
    }

Type: 'function_call'
---------------------
Example: function()
Structure:
    {
        'type': 'function_call'
    }

Type: 'function_def'
--------------------
Example: function(){ ... } (complete)
Structure:
    {
        'type': 'function_def',
        'body': [...],
        'line': 5
    }

Type: 'window_pack'
-------------------
Example: window.pack
Structure:
    {
        'type': 'window_pack'
    }

Type: 'variable'
----------------
Example: x = "hello"
Structure:
    {
        'type': 'variable',
        'name': 'x',
        'value': '"hello"',
        'line': 1
    }

Type: 'unknown'
---------------
Example: invalid_syntax_here
Structure:
    {
        'type': 'unknown',
        'raw': 'invalid_syntax_here',
        'line': 1
    }

FUNCTION: _extract_method_params(line)
--------------------------------------

Signature:
    def _extract_method_params(line):

Purpose:
    Extract window_name, method_name, and params_str from a window method call.
    Handles nested parentheses like onClick:function().

How it works:
    1. Matches window.method( pattern
    2. Counts parenthesis depth to find matching closing )
    3. Extracts parameters string between parentheses

Returns:
    tuple: (window_name, method_name, params_str) or None

Examples:
    _extract_method_params('window.Text(text:"hi", x:100)')
    → ('window', 'Text', 'text:"hi", x:100')
    
    _extract_method_params('window.Button(onClick:function(), x:100)')
    → ('window', 'Button', 'onClick:function(), x:100')


FUNCTION: _split_params(params_str)
-----------------------------------

Signature:
    def _split_params(params_str):

Purpose:
    Split a parameter string by commas, but ignore commas inside parentheses.

How it works:
    1. Iterates through each character
    2. Tracks parenthesis depth
    3. Only splits on commas when depth == 0

Returns:
    list: Parameter pairs

Examples:
    _split_params('onClick:function(), x:100, y:200')
    → ['onClick:function()', 'x:100', 'y:200']
    
    _split_params('text:"Hello, World!", x:100')
    → ['text:"Hello, World!"', 'x:100']


FUNCTION: parse_statement(line)
-------------------------------

Signature:
    def parse_statement(line):

Purpose:
    Parse a single statement line

Parameters:
    line (str): A single line of code

Returns:
    dict or None: Statement info if valid, None if empty/comment


FUNCTION: parse_all_statements(content)
---------------------------------------

Signature:
    def parse_all_statements(content):

Purpose:
    Parse all statements from file content, including function bodies

Parameters:
    content (str): Full file content

Returns:
    list: All statements with line numbers

How it works:
    1. Splits content into lines
    2. Tracks function start/end
    3. Collects function body statements
    4. Returns flat list with function_def containing body

================================================================================
                      NESTED PARENTHESES SUPPORT
================================================================================

The parser correctly handles:

✅ onClick:function()
✅ colourRGB:255, 0, 0
✅ text:"Hello, World!"
✅ Multiple nested: callback:function(a, b)

Example:
    window.Button(onClick:function(), x:100, y:200, colour:green)
    
    Parsed as:
    - onClick: function_call
    - x: 100 (number)
    - y: 200 (number)
    - colour: green (named_color)
