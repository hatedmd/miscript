================================================================================
                            QUICK REFERENCE
================================================================================

SYNTAX HIGHLIGHTS:
- Variables: var_name = "value"
- Input: name = input("prompt > ")
- Print: print(text)
- Window: window = window.init(600, 600)
- Text: window.Text(text:$var, x:100, y:100)
- Background: window.BG(colour:green)
- Input Field: window.Input(entry:var, x:100, y:200)
- Button: window.Button(onClick:function(), x:100, y:250, colour:green)
- Functions: function(){ ... }
- Refresh: window.pack

VARIABLE SYNTAX:
- Plain: $variable or variable
- Interpolation: "Hello $name." (space after $name is consumed)
- Entry Keyword: entry (gets input field value)

COLOR FORMATS:
- RGB: window.BG(colourRGB:255, 0, 0)
- HEX: window.BG(colourHEX:#FF0000)
- Named: window.BG(colour:red)

WINDOW METHODS:
- Text: Display text label
- BG: Set background color
- Input: Create text input field
- Button: Create clickable button with onClick
- pack: Refresh/update window display

SAFE BUILTINS:
str(), int(), float(), len(), abs(), round(), min(), max(), sum()
